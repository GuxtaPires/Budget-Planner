# BudgetPlanner
## Organize suas finanças de uma maneira divertida!
Este sistema foi feito para gerenciamento de finanças, utilizando 4 métodos:
- Entrada de dinheiro;
- Saída de dinheiro;
- Investimento;
- Retirada de investimento.
